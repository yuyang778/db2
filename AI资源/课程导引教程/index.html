核心知识

数据库基本概念

数据库 (DB)

数据库是一个存储数据的"仓库"，它保存了一系列有组织的数据，是按照数据结构来组织、存储和管理数据的集合。

数据库管理系统 (DBMS)

数据库是通过DBMS创建和操作的容器。简称数据库软件或数据库产品，常常被直接称为数据库。

SQL

结构化查询语言(Structure Query Language)，是专门用来与数据库通信的标准语言，几乎所有关系型数据库都支持。

学生成绩数据库范例

存储哪些信息？

  • 成绩信息
  • 学生信息
  • 课程信息

每类信息包含哪些数据？

学生:

学号、姓名、性别、密码、专业

课程:

课程号、课程名、学分

成绩:

学号、课程号、成绩

以什么形式保存？

二维表（关系）

数据以表格形式存储，每一行表示一条记录

数据是如何存储的？

数据是以二维表的形式来存放，将数据放到表中，表再放到库中。

二维表在数据库中又叫关系，所以又称为关系数据库。

SQL Server：RDBMS

关系数据库管理系统

  • 一个数据库包含多张表，表和表之间是相互关联的
  • 每个表都有一个名字，用来标识自己，表名在数据库中具有唯一性
核心理念

为什么要用数据库？

保存数据的其他方式

数组 集合 文件 其他方式

传统数据存储方式在数据量大、结构复杂、需要多人访问时面临诸多限制。

数据库特点

  • 数据结构化，数据之间具有联系，面向整个系统
  • 数据的共享性高，冗余度低，易扩充
  • 数据独立性高
  • 数据由DBMS统一管理和控制
